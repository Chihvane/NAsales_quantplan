# Part 4 Unified Chain

## 定位

`Part 4` 是销售路径、渠道效率与 ROI 决策层，负责把市场机会和供应能力转成具体渠道进入方案。

## 核心问题

- 先卖哪个渠道
- 独立站 / 平台 / 线下 B2B 哪条路径最成立
- 获客、转化、退货与费用一起看后 ROI 是否站得住
- 何时试点、何时放大、何时回滚

## 统一链条

### 1. 策略层

固定覆盖：

- 独立站模式
- 平台电商模式
- 线下经销 / ToB 模式
- 曝光与获客方式
- ROI 与投产模型
- 最优进入路径

### 2. 量化层

固定输出：

- 渠道级 P&L
- Go / No-Go 门禁
- 获客与留存指标
- 实验设计
- ROI 蒙特卡洛情景
