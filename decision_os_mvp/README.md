# Decision OS MVP

Minimal executable MVP for `Decision OS v3.0`.

## Files

- `sample_data.py`
- `models.py`
- `capital_risk.py`
- `gate_engine.py`
- `main.py`
- `test_mvp.py`

## Run

```bash
python3 decision_os_mvp/main.py
```

## Test

```bash
python3 -m unittest decision_os_mvp.test_mvp -v
```
